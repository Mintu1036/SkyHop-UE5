# 🎮 SkyHop – Unreal Engine 5 Platformer Game

## 🚀 Overview

**SkyHop** is a simple 3D platformer game developed using Unreal Engine 5.
The player navigates across platforms, avoids falling, and reaches the finish point within a limited time.

This project was created as part of my learning journey in Unreal Engine, focusing on core gameplay mechanics and Blueprint scripting.

---

## 🎯 Gameplay

* Move and jump across platforms
* Avoid falling off the level
* Complete each level before the timer runs out
* Progress through multiple levels

---

## 🧩 Features

* 🕹️ Third-person character movement
* ⏱️ Countdown timer system
* 🧱 Platform-based level design
* 🎮 Level selection UI
* 💾 Save system for progress
* 🎨 Basic UI with Blueprint logic

## ⚙️ How to Play

1. Launch the game (.exe file)
2. Select a level from the main menu
3. Use movement controls to navigate
4. Reach the end before the timer reaches zero

---

## 🎮 Controls

* **W / A / S / D** → Move
* **Mouse** → Look around
* **Spacebar** → Jump
